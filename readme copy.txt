== Installation ==

**Automatic Installation:**

1. Go to your WordPress Dashboard
2. Navigate to **Plugins > Add New > Upload Plugin**
3. Upload the `rk-photo-gallery-block.zip` file
4. Click **Install Now**, then **Activate**

**Manual Installation:**

1. Download the plugin ZIP file
2. Extract the contents to `/wp-content/plugins/rk-photo-gallery-block/`
3. Go to the WordPress admin panel and activate the plugin

== Usage ==

1. Create or edit a post or page
2. Open the Gutenberg Block Editor
3. Click the "+" button and search for **RK Photo Gallery Block**
4. Add images from your media library or upload new ones
5. Configure layout, image size, and other settings from the block sidebar

== Frequently Asked Questions ==

= Does this plugin support the Classic Editor? =  
No. RK Photo Gallery Block is built specifically for the Gutenberg (Block) Editor.

= Is it compatible with all themes? =  
Yes, the plugin works with any theme that supports Gutenberg blocks.

= How can I access Pro features? =  
You will need to upgrade to the Pro version to unlock Carousel, Lightbox, and additional features.

= Can I define custom image sizes? =  
Yes. The plugin allows selecting predefined sizes or setting a custom width and height.
